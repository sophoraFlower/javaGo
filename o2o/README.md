## 项目介绍

        项目运用SSM快速迭代出校园商铺1.0，再结合SSM转型成Spring Boot的思路，切换到校园商铺2.0。同时包含MySQL主从同步实现读写分离，利用SUI Mobile快速实现响应式页面，Redis缓存，阿里云部署等实用技术点。
